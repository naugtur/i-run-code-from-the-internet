# i-run-code-from-the-internet

A demo of the security provided by LavaMoat.  
The `demo` branch contains a sequence of commits with various versions of a "useful" package turning malicious.

---

Demo setup

```
git switch demo
git checkout 1
node sink/index.mjs &
PS1='>'
```

Demo playthrough

```
gitnext
npm start
npm test
```
